TOPIC
    about_GPRegistryPolicyDsc

SHORT DESCRIPTION
    DSC resources to apply and manage local group policies.

LONG DESCRIPTION
    This resource module contains DSC resources used to apply and manage local group policies by modifying the respective .pol file.

EXAMPLES
    PS C:\> Get-DscResource -Module GPRegistryPolicyDsc

    writing code, sharing opinions, and provided feedback.
