SHORT DESCRIPTION
    DSC resources for managing certificates on a Windows Server.

LONG DESCRIPTION
    This module contains DSC resources for managing certificates on a Windows
    Server.

EXAMPLES
    PS C:\> Get-DscResource -Module CertificateDsc
